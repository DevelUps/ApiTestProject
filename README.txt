Manual de Uso para Ejecutar las Pruebas del Proyecto apiTestProject
Requerimientos
Para poder ejecutar las pruebas del proyecto apiTestProject, se necesita tener instalado lo siguiente:

Visual Studio
.NET Core
Configuración
Clonar el repositorio de GitHub del proyecto apiTestProject.
Abrir la solución apiTestProject.sln en Visual Studio.
Compilar la solución para restaurar los paquetes necesarios y asegurarse de que no hay errores de compilación.
En el Explorador de soluciones, expandir la carpeta BreweryTests.
Hacer clic derecho en el archivo BreweryTests.cs y seleccionar Run Tests en el menú contextual.
Ejecución
Para ejecutar las pruebas del proyecto apiTestProject, siga los siguientes pasos:

Abra Visual Studio y cargue la solución apiTestProject.sln.
Asegúrese de que no hay errores de compilación.
En el Explorador de soluciones, expanda la carpeta BreweryTests.
Haga clic derecho en el archivo BreweryTests.cs y seleccione Run Tests en el menú contextual.
El resultado de las pruebas se mostrará en la ventana Test Explorer.
Si alguna de las pruebas falla, se mostrará un mensaje indicando cuál es la causa del error y en qué punto exacto de la prueba ocurrió. De esta manera, podrá depurar y corregir el problema para que las pruebas puedan ser ejecutadas exitosamente.
